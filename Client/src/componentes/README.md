# Componentes Visuales

Esta carpeta contendrá todos los componentes visuales organizados por dominio:

- `comun/`: Componentes genéricos y reutilizables.
- `tienda/`: Componentes específicos del catálogo y filtros.
- `productos/`: Cards de producto, fichas y galerías.
- `carrito/`: Items del carro y resúmenes de compra.
- `checkout/`: Formulario de envío y pasarela visual.
- `administracion/`: Tablas de TanStack Table y modales de edición.
- `layout/`: Cabeceras, pies de página y menús laterales.
